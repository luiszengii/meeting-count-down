# ADRs

ADR 表示 Architecture / Decision Record，用于记录技术栈、产品形态、接入策略、文档规则和架构方向的变化。ADR 的目标不是复盘聊天，而是沉淀“为什么当时这么决定”。

## 命名规则

- `YYYY-MM-DD-kebab-case-title.md`
